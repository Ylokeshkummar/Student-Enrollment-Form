<h3>Description</h3>
<p>Major responsibility of the project is to develop the javascripts to create putrequest, getrequest and update request etc.
 Created a form named Student Enrollment Form
Form included with three buttons named save,update and reset
After entering the data in the form after clicking on the save button the data given on the form will be accepted and data will be stored in JPDB
